# CAMS — Claude Agent Management System

This directory is the source of truth for in-flight and finished work. See **CLAUDE.md §4** for the full protocol.

- `active/` — one markdown file per open task (`YYYY-MM-DD_<slug>.md`).
- `completed/` — tasks finished, with an **Outcome** section appended.

**Before creating a task, the PM MUST read every file in `active/` and `completed/`** to avoid duplicates and regressions.

Use the task template in CLAUDE.md §4. Do not start work that isn't represented by a task file here.
